# Image Gallery

A simple and responsive Image Gallery built using **HTML, CSS, and JavaScript** as part of my **CodeAlpha Frontend Development Internship**.

## ✨ Features

- Responsive image gallery
- Previous and Next buttons
- Lightbox image view
- Hover effects
- Smooth transitions

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript

## ▶️ How to Run

1. Download or clone this repository.
2. Open the project folder.
3. Open `index.html` in your browser.

## 📚 What I Learned

- Creating responsive layouts
- Using CSS hover effects
- JavaScript DOM manipulation
- Event handling
- Building interactive web pages
