# Calculator

A simple Calculator built using **HTML, CSS, and JavaScript** as part of my **CodeAlpha Frontend Development Internship**.

## ✨ Features

- Addition, Subtraction, Multiplication, and Division
- Clear button
- User-friendly interface
- Responsive design
- Real-time calculations

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript

## ▶️ How to Run

1. Download or clone this repository.
2. Open the project folder.
3. Open `index.html` in your browser.
